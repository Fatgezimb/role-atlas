# Design References

These accepted concepts are the visual source of truth for the first Role Atlas implementation:

- `role-atlas-primary-map.png`: map-first search, filters, role list, analytics, and marker states.
- `role-atlas-job-detail.png`: selected-role modal/drawer with remote classification, license eligibility, benefits, provenance, and market insights.

The UI implementation should keep the same product posture: true white/cool-gray surfaces, crisp dense controls, teal for remote jobs, orange for onsite/hybrid jobs, thin borders, compact cards, and a map-first workflow.
